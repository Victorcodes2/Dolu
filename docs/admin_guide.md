## Managing Parcels

### Adding a New Parcel
1. Click "Add New Parcel" button
2. Fill in all required fields:
   - Tracking ID (unique identifier)
   - Sender details
   - Receiver details
   - Package details
   - Expected delivery date
3. Click "Add Parcel" to save

### Updating Parcels
1. Click the edit icon next to a parcel
2. Modify the required fields
3. Click "Update Parcel" to save changes

### Deleting Parcels
1. Click the delete icon next to a parcel
2. Confirm deletion in the prompt

### Parcel Statuses
- **Awaiting Pickup**: Initial status when parcel is registered
- **In Transit**: Parcel is in the delivery network
- **Out for Delivery**: Final delivery attempt in progress
- **Delivered**: Successfully delivered to recipient
- **Delayed**: Delivery is experiencing delays
- **Cancelled**: Delivery has been cancelled

## Managing Messages

### Viewing Messages
- Unread messages are highlighted
- Click "Mark Read" to mark a message as read
- Click "Mark Unread" to mark a message as unread

### Deleting Messages
- Click the delete icon next to a message
- Confirm deletion in the prompt

## Search and Sort
- Use the search bar to filter parcels or messages
- Click column headers to sort by that field
- Click again to toggle between ascending and descending order

## Security Notes
- Always log out when leaving the admin panel
- Don't share the admin password
- Regularly check for unauthorized access
